Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usandoa tag "bold {b}" (não semantica)

Nesta frase eu tenho um um termo em destaque usando a tag "Strong" (semântica)

Italico / Enfase

Nesta farase temos um termo em italico utilizando a tag "I" (não semântica)

Nesta frase, temos um termo em enfase usando a tag "EM" ( semântica)

Texto marcado

Podemos tambem criar um texto marcado usando a tag "MARK"

Texto grande e pequeno

estamos crindo um texto grande e um texto pequeno nesse parágrafo usando "big" e "small" como tags

Texto excluido ou deletado

Podemos marcar um texto como apagado para dar enfase e indicar que foi apagado usando a tag "del"

Texto inserido

Podemos marcar um texto como inserido para dar enfase que o tal foi adicionado usando a tag "ins" ou "u"

Texto sobrescrito

para inserir coisa do tipo x20+3 usando a tag "sup"

Texto subscrito

Para inserir coisas como H20 usandoa tag "sub"
